# 🩺 Diabetes Risk Prediction

A Data Analysis project that explores diabetes-related health factors using Python, Pandas, NumPy, Matplotlib, and Seaborn.

---

## 📌 Project Overview

This project focuses on analyzing patient health records to understand patterns and relationships associated with diabetes. The project includes data preprocessing, feature engineering, exploratory data analysis (EDA), and data visualization.

---

## 🎯 Objectives

- Analyze patient health data
- Perform data cleaning and preprocessing
- Create meaningful health categories
- Explore patterns using EDA
- Generate insightful visualizations

---

## 🛠 Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Project Workflow

### 1. Data Preprocessing

- Handling missing values
- Removing duplicates
- Data type conversion
- Feature engineering

---

### 2. Feature Engineering

Created health-related categories including:

- Gender Category
- Education Category
- Smoking Category
- Cholesterol Category
- Systolic Blood Pressure Category
- Diastolic Blood Pressure Category
- BMI Category
- Heart Rate Category
- Diabetes Category
